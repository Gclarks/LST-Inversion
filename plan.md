# Landsat 8 单通道算法地表温度反演系统 — 实施计划

## 1. 项目概述与目标

基于 Landsat 8 OLI/TIRS 影像，使用单通道算法（Single-Channel Algorithm）反演地表温度（LST），并提供图形化操作界面。

### 核心技术链路

```
Landsat 8 影像目录
    │
    ├─ MTL.txt ───────────→ 元数据解析（定标系数、太阳高度角、成像时间等）
    │
    ├─ Band 4 (Red) ──────→ DN → TOA Reflectance ──┐
    ├─ Band 5 (NIR) ──────→ DN → TOA Reflectance ──┤─ NDVI ─→ 地表比辐射率(ε)
    │                                                  │
    ├─ Band 10 (TIR) ─────→ DN → TOA Radiance ────────┤
    │                                                  │
    ├─ 水汽含量 w ─────────────────────────────────────┤
    │   (MODIS / 手动输入 / 估算)                       │
    │                                                  │
    └─ 单通道算法 ────→ B(Ts) ────→ 地表温度 Ts ──→ 输出 GeoTIFF
```

## 4. 模块详细设计

### 4.1 文件识别与元数据解析（`utils/file_utils.py` + `core/metadata.py`）

**目录扫描逻辑：**
- 遍历用户选择的输入目录
- 按文件名模式匹配各波段：`*_B1.TIF` ~ `*_B11.TIF`（Landsat 8 Collection 2 命名）
- 按扩展名 `.txt` 匹配 MTL 元数据文件
- 将识别的波段路径存入字典 `{band_number: filepath}`

**MTL.txt 解析：**
- 解析 `GROUP = ... / END_GROUP = ...` 结构
- 提取关键参数：
  - `RADIANCE_MULT_BAND_10` / `RADIANCE_ADD_BAND_10`（热红外辐射定标系数）
  - `REFLECTANCE_MULT_BAND_4/5` / `REFLECTANCE_ADD_BAND_4/5`（反射率定标系数）
  - `SUN_ELEVATION`（太阳高度角）
  - `DATE_ACQUIRED` / `SCENE_CENTER_TIME`（成像时间）
  - `EARTH_SUN_DISTANCE`（日地距离，用于大气层顶太阳辐照度计算）
- 元数据弹窗以只读表格形式展示

### 4.2 辐射定标（`core/calibration.py`）

**多光谱波段（Band 4 Red, Band 5 NIR）— DN → TOA Reflectance：**

$$ρ_λ' = M_ρ × DN + A_ρ$$

$$ρ_λ = \frac{ρ_λ'}{\sin(θ_{SE})} = \frac{ρ_λ'}{\cos(θ_{SZ})}$$

其中：
- $M_ρ$ = `REFLECTANCE_MULT_BAND_n`
- $A_ρ$ = `REFLECTANCE_ADD_BAND_n`
- $θ_{SE}$ = 太阳高度角（从 MTL 获取）
- ρ_λ 输出范围 0~1，超出范围的做 clip 处理

**热红外波段（Band 10）— DN → TOA Radiance：**

$$L_λ = M_L × DN + A_L$$

其中：
- $M_L$ = `RADIANCE_MULT_BAND_10`
- $A_L$ = `RADIANCE_ADD_BAND_10`
- 输出单位：W·m⁻²·sr⁻¹·μm⁻¹

### 4.3 NDVI 与地表比辐射率（`core/emissivity.py`）

**NDVI 计算：**

$$NDVI = \frac{NIR - Red}{NIR + Red} = \frac{ρ_{Band5} - ρ_{Band4}}{ρ_{Band5} + ρ_{Band4}}$$

**植被覆盖度（FVC）：**

$$P_v = \left( \frac{NDVI - NDVI_{soil}}{NDVI_{veg} - NDVI_{soil}} \right)^2$$

- $NDVI_{soil}$ = 0.2（裸土 NDVI 阈值）
- $NDVI_{veg}$ = 0.5（浓密植被 NDVI 阈值）
- $P_v$ 限制在 [0, 1]

**地表比辐射率 ε（Sobrino et al. 2004）：**

| NDVI 范围 | 公式 |
|-----------|------|
| NDVI < 0.2 | ε = 0.973 - 0.047 × ρ_red（裸土/水体） |
| 0.2 ≤ NDVI ≤ 0.5 | ε = ε_v × P_v + ε_s × (1-P_v) + Δε（混合像元） |
| NDVI > 0.5 | ε = 0.99（浓密植被） |

混合像元参数（Landsat 8 Band 10）：
- ε_v = 0.986（植被比辐射率）
- ε_s = 0.972（土壤比辐射率）
- Δε = 0（忽略腔体效应项，或取 (1-ε_s) × (1-P_v) × F × ε_v，F = 0.55）

### 4.4 水汽含量（`core/water_vapor.py`）— 弱优先级

三种获取方式，按优先级：

1. **手动输入**（默认）：用户在 GUI 中输入水汽含量（g/cm²），此值也可从 NCEP/NCAR 再分析数据或气象站点数据获取
2. **默认值估算**：若无输入，使用典型中纬度夏季大气廓线估算（w ≈ 1.5 ~ 3.0 g/cm²）
3. **MODIS 自动下载**（选做）：通过 NASA LAADS DAAC API 下载 MOD05_L2 或 MOD07_L2 近实时产品，裁剪至研究区域，取均值作为 w

### 4.5 单通道算法反演地表温度（`core/lst_inversion.py`）

**步骤 1：计算 B(Ts)，即大气校正后的地表黑体辐亮度**

$$B(T_s) = a_0 + a_1 w + (a_2 + a_3 w + a_4 w^2)\frac{1}{\epsilon} + (a_5 + a_6 w + a_7 w^2)\frac{1}{\epsilon}L_{sen}$$

其中：
- $w$ = 大气水汽含量（g/cm²）
- $\epsilon$ = 地表比辐射率
- $L_{sen}$ = Band 10 的 TOA Radiance（步骤 4.2 结果）
- $a_0$ ~ $a_7$ = 算法系数

**算法系数 a₀~a₇（按水汽含量 w 分段）：**

系数通过 MODTRAN 辐射传输模型对标准大气廓线（TIGR 数据库）模拟拟合得到。根据水汽含量 w（g/cm²）选择对应系数组：

| 系数 | w ∈ [0, 2.0] | w ∈ [2.0, 4.0] | w ∈ [4.0, 7.0] | Full range（不分段） |
|------|-------------|---------------|---------------|---------------------|
| a₀ | -0.28009 | -0.60336 | 2.280539 | -0.4107 |
| a₁ | 1.257429 | 1.613485 | 0.918191 | 1.493577 |
| a₂ | 0.275109 | -4.98989 | -38.3363 | 0.278271 |
| a₃ | -1.32876 | 2.772703 | 13.82581 | -1.22502 |
| a₄ | -0.1696 | -1.04271 | -1.75455 | -0.31067 |
| a₅ | 0.999069 | 1.739598 | 5.003919 | 1.022016 |
| a₆ | 0.033453 | -0.54978 | -1.62832 | -0.01969 |
| a₇ | 0.015232 | 0.129006 | 0.196687 | 0.036001 |

**系数选择策略**：按用户输入的 w 值落入的区间自动选组。若 w 超出分段范围（<0 或 >7.0），使用 Full range 系数组作为回退。

**步骤 2：Planck 反函数求地表温度**

$$T_s = \frac{c_2 / \lambda}{\ln\left(\frac{c_1}{\lambda^5 \cdot B(T_s)} + 1\right)}$$

其中：
- $c_1$ = 1.19104 × 10⁸ W·μm⁴·m⁻²·sr⁻¹（第一辐射常数）
- $c_2$ = 1.43877 × 10⁴ μm·K（第二辐射常数）
- $\lambda$ = 10.9 μm（Landsat 8 Band 10 有效波长）
- 输出温度单位：K（界面可切换显示为 ℃）

  - `_temp_/toa_radiance.tif`（Band 10 TOA Radiance）
  - `_temp_/toa_reflectance_b4.tif`（Band 4 TOA Reflectance）
  - `_temp_/toa_reflectance_b5.tif`（Band 5 TOA Reflectance）
  - `_temp_/ndvi.tif`
  - `_temp_/emissivity.tif`
- 最终 LST 输出后自动删除 `_temp_` 目录及其内容

## 9. 关键风险与缓解

| 风险 | 缓解措施 |
|------|---------|
| 水汽值 w 恰好落在分段边界（2.0, 4.0）时系数不连续 | 按 "左闭右开" 约定：w < 2.0 用第一组，2.0 ≤ w < 4.0 用第二组，w ≥ 4.0 用第三组；边界附近两组都算并给出差异提示 |
| 大影像内存溢出 | GDAL 支持分块读取，按 block 逐行处理 |
| GDAL 写入 GeoTIFF 丢失投影信息 | 从原始 Band 10 影像复制投影/地理变换参数 |
| 不同 Landsat 8 Collection 版本 MTL 格式差异 | 兼容 Collection 1 与 Collection 2 的 key 命名差异 |
